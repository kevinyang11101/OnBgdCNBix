# 旅游信息管理系统 java348

## 项目概述

旅游信息管理系统是一款基于B/S架构和MVC设计模式的Web应用程序。它采用前后端分离的开发模式，前端使用Vue.js框架，后端采用Java和SpringBoot技术栈。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**: 
  - IDEA
  - JDK 17
  - Maven
  - Mysql
- **技术栈**: 
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue
  - Redis

## 功能模块

### 登录模块

- 用户登录
- 用户注册
- 忘记密码

### 用户前台

- 首页
- 景点信息（预约）
- 旅游路线
- 景区酒店（预约）
- 旅游咨询
- 我的收藏
- 我的预定

### 后台管理

- 预约统计图
- 景点管理
- 线路管理
- 酒店管理
- 咨询管理
- 景点预约管理
- 酒店预约管理
- 轮播图管理
- 用户管理

## 系统角色

系统主要包括两大角色：普通用户和管理员。

- **普通用户**: 可以查看旅游信息、预约景点和酒店、收藏感兴趣的旅游信息等。
- **管理员**: 负责管理后台，包括景点、路线、酒店、用户等信息的维护与更新。

## 运行环境

- 前端：现代浏览器
- 后端：JDK 17及以上版本，Mysql数据库，Redis缓存服务器

## 核心亮点

- **前后端分离**: 提高开发效率，降低前后端耦合度。
- **高效缓存机制**: 使用Redis作为缓存，提高系统性能。
- **完善的权限管理**: 确保系统安全，不同角色访问控制精准。
